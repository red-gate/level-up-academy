# Stringly-typing workshop - example solution

This folder contains an example solution for the [stringly-typing workshop](../stringly-typing/README.md).

The main program takes two command-line arguments, the path to the input purchases file and the common currency to convert each purchase to. We recommend setting the following as the program arguments when running the application via your IDE, otherwise you'll get a usage error message.

```
..\..\..\..\PurchasedItems.csv USD
```
